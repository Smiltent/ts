# ts
Created for the ts YSWS for Hackclub.
## what is this project?
An entire whiteboard, where admins can draw and save to the database, where everyone can see it on the page.
## running
On Linux, you must have [bun](https://bun.sh) installed. 
### linux / other OS
```bash
bun run start
```
### docker (compose)
```bash
sudo docker compose up -d
```
